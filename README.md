# 🔥 Passion Play — PWA para Parejas

App de juegos íntimos para parejas. Funciona como PWA instalable directamente desde el navegador.

## 🎮 Juegos incluidos

- 🎡 **Ruleta de Posiciones** — Rueda animada con 5 posiciones por nivel
- 🎲 **Dados del Deseo** — Combina acción + zona + duración
- 🃏 **Truth or Dare** — Verdad o reto, 3 niveles de intensidad
- 🎰 **Slots de Pasión** — 3 reeles: acción + lugar + duración

## 🚀 Cómo publicar en GitHub Pages

### Paso 1 — Crear repositorio
1. Ve a [github.com](https://github.com) e inicia sesión
2. Click en **"New repository"**
3. Nombre: `passion-play` (o el que quieras)
4. Marca como **Public**
5. Click **"Create repository"**

### Paso 2 — Subir los archivos
Arrastra y suelta estos archivos/carpetas en GitHub:
```
index.html
manifest.json
sw.js
icons/
  icon-72x72.png
  icon-96x96.png
  icon-128x128.png
  icon-144x144.png
  icon-152x152.png
  icon-192x192.png
  icon-384x384.png
  icon-512x512.png
```

### Paso 3 — Activar GitHub Pages
1. Ve a **Settings** del repositorio
2. Sección **Pages** en el menú izquierdo
3. En **Source** selecciona: `Deploy from a branch`
4. Branch: `main` → folder: `/ (root)`
5. Click **Save**

### Paso 4 — Acceder a tu app
En 1-2 minutos estará disponible en:
```
https://TU-USUARIO.github.io/passion-play/
```

## 📲 Instalar en el celular

### Android (Chrome)
1. Abre la URL en Chrome
2. Aparecerá banner automático "Agregar a pantalla de inicio"
3. O toca los 3 puntos → "Instalar app"

### iPhone (Safari)
1. Abre la URL en Safari
2. Toca el botón de compartir (cuadrado con flecha)
3. Selecciona **"Agregar a pantalla de inicio"**

## ✨ Características
- Sin registro ni cuenta
- Todo privado, datos solo en tu dispositivo
- Funciona sin internet después de la primera visita
- 3 niveles: Romántico 🌹 / Picante 🌶️ / Hot 🔥
